SYNAPSE SEARCH - RELEASE
========================

Requirements:
- Node.js installed (https://nodejs.org/)

Installation & Running:
1. Double-click the "install_and_run.command" file.
2. The script will automatically:
   - Check for Node.js
   - Install all necessary libraries
   - Start the server
   - Open your web browser to the application

Note: 
If you get a security warning on Mac about the script, you may need to:
1. Right-click "install_and_run.command"
2. Select "Open"
3. Click "Open" in the dialog box.

Fresh Start:
This release does not include any previous usage data. It will create a fresh database file (news.db) automatically when you run it for the first time.
